# AI Digital Product Studio Dashboard

Bu repo, AI destekli dijital ürün üretim ve satış planını görsel bir web dashboard olarak sunar.

## Teknoloji

- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide Icons

## Lokal Çalıştırma

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## GitHub'a Push

```bash
git init
git add .
git commit -m "Initial dashboard"
git branch -M main
git remote add origin https://github.com/atpmaster/ai-digital-product-studio.git
git push -u origin main
```
